# kopterix-deposit

Convenience mirror of the Kopterix validation deposit: a passive
longitudinal observational instrument for monitoring emergent behavior
in agent-native multi-agent social platforms. The files here are the
unpacked contents of the archived deposit, provided so readers can
browse them without downloading and unzipping the archive.

## Repository-level information (not part of the archived record)

Everything in this section is repository metadata added for
convenience. It is not part of the deposit and does not appear in the
archived files.

## One deviation from the archive

The deposit contains a file named README.md. In this repository that
file appears as README_deposit.md, renamed so that this repository
level page could occupy the default README position. Its contents are
unaltered. This is the only difference in file naming between the
archive and this repository, and no file contents were changed.
